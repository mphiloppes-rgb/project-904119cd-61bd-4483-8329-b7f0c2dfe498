

# اقتراحات لتطوير وتمييز نظام "الراعي"

## الحالة الحالية
المشروع فيه: نقطة بيع، منتجات، عملاء، موردين، فواتير شراء/بيع، مرتجعات، مصاريف، تقارير، نسخ احتياطي. شغال بـ localStorage.

## أهم الاقتراحات (مرتبة بالأولوية)

### 1. تحسينات تشغيلية مهمة (سريعة وقيمة عالية)
- **Barcode / كود سريع في POS**: حقل بحث يقرأ الباركود تلقائياً ويضيف المنتج للسلة بضغطة Enter — يسرّع البيع جداً.
- **اختصارات لوحة المفاتيح**: F2 لفاتورة جديدة، F3 للبحث، F4 للدفع، Esc للإلغاء.
- **طباعة إيصال حراري 80mm**: قالب طباعة مخصص للطابعات الحرارية بجانب الـ A4 الموجود.
- **تنبيه نقص المخزون في الداشبورد**: كارت يعرض المنتجات اللي وصلت للحد الأدنى مع زر "اطلب من المورد".

### 2. تمييز تجاري (يخلي المنتج أقوى من المنافسين)
- **حسبة الربح اللحظية في POS**: أثناء البيع يظهر للكاشير صافي ربح الفاتورة (مخفي عن العميل) — مفيد للتفاوض.
- **أسعار متعددة للمنتج**: سعر جملة / نصف جملة / قطاعي، يتغير تلقائي حسب الكمية.
- **خصومات مرنة**: خصم بالنسبة % أو بالمبلغ، على الفاتورة كلها أو على صنف معين، مع تسجيل سبب الخصم.
- **عرض كشف حساب العميل/المورد**: شاشة فيها كل الفواتير والمدفوعات بترتيب زمني + رصيد متحرك (ميزان مراجعة).

### 3. أمان البيانات (نقطة ضعف حالية)
- **النظام كله بـ localStorage** = أي مسح للمتصفح = ضياع كل البيانات. اقتراحات:
  - **نسخ احتياطي تلقائي يومي** ينزل ملف JSON على الكمبيوتر (في نسخة Electron).
  - **تذكير بالنسخ الاحتياطي** كل أسبوع لو المستخدم نسي.
  - **ترقية لـ Lovable Cloud** (Supabase) — يحفظ السحابة + multi-device + مستخدمين متعددين.

### 4. صلاحيات وحماية
- **PIN code للكاشير**: كل مستخدم له كود، والمدير بس يقدر يشوف التقارير والأرباح ويعمل مرتجعات.
- **سجل العمليات (Audit log)**: مين عمل إيه وامتى (مرتجع، خصم، حذف منتج).

### 5. تحسينات تقارير
- **Charts بصرية**: رسم خطي لمبيعات آخر 30 يوم، Pie chart لأكثر المنتجات مبيعاً، Bar chart للأرباح الشهرية.
- **مقارنة فترات**: "هذا الشهر مقابل الشهر السابق" مع نسبة التغير %.
- **تقرير العملاء الأكثر شراءً** + **تقرير المنتجات الراكدة** (ملهاش حركة من X يوم).

### 6. تجربة استخدام
- **Dark mode** — مفيد للاستخدام الليلي في المحلات.
- **بحث موحد (Command palette)** بـ Ctrl+K يبحث في كل حاجة (منتج، عميل، فاتورة).
- **اللغة**: زرار تبديل عربي/إنجليزي.

## اقتراحي للمرحلة القادمة

أهم 3 حاجات لو هتختار:
1. **Barcode + اختصارات + طباعة حرارية** (تشغيلي مهم جداً)
2. **كشف حساب عميل/مورد + أسعار جملة/قطاعي** (تمييز تجاري)
3. **نسخ احتياطي تلقائي أو ترقية Lovable Cloud** (حماية بيانات)

اختار اللي يهمك وأبدأ تنفيذه.
